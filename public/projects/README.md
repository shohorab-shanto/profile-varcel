## Image Recommendations

- **Size**: 800x600 pixels or 1200x800 pixels (16:9 or 4:3 ratio)
- **Format**: JPG or PNG
- **Quality**: High quality screenshots work best
- **Content**: Show the main interface/feature of each project

## How to Add Images

1. Take screenshots of your projects
2. Resize them to recommended dimensions
3. Rename them according to the list above
4. Place them in this `public/projects/` folder
5. The images will automatically appear on your portfolio

## Temporary Solution

If you don't have screenshots ready, you can use placeholder images from:
- https://unsplash.com (search for relevant tech/business images)
- Create simple gradient backgrounds with project names
- Use project logos if available

The portfolio will work with or without images - it shows a fallback icon if an image is missing.
